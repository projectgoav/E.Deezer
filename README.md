# E.Deezer
An unoffical asynchronous wrapper for the Deezer API and .NET.

## Usage

[Via Nuget](https://www.nuget.org/packages/E.Deezer)

From source:
```
git clone https://github.com/projectgoav/e.deezer
```

See E.Deezer.Examples for C# example usage. *(Source only)*


## Features
- Searching
- Service Information
- Variable result sizes

**Object Methods**
- Album
	- Artist
	- Tracklist
- Artist
	- Albums
	- Top Tracks
	- Tracklist
	- Related Artists
	- Featured Playlists
	
- Playlist
	- Tracklist
	
- Track
	- Artist
	- Album
	
- User
	- Current User
	- Favourite Playlists 
	

*More features later*

## Requirements
- Visual Studio 2012 (or later)
- [RestSharp](http://restsharp.org/)
